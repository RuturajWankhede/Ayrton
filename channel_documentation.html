Telemetry CSV Format Guide

Channel naming conventions and data format documentation

Intelligent Channel Detection

The system automatically detects your column names. Only Time, Distance, and Speed are required!

Required Channels (Minimum 3)

Time

Elapsed time since session start

Accepted Names:

Time time Elapsed Time Session Time t

Unit:

Seconds (e.g., 45.123)

Example: 87.654

Distance

Distance traveled in current lap

Accepted Names:

Distance Lap Distance Dist LapDist

Unit:

Meters (e.g., 1234.5)

Example: 2847.3

Speed

Vehicle speed

Accepted Names:

Speed Ground Speed Vehicle Speed GPS Speed

Unit:

km/h or mph

Example: 185.7

Basic Channels (Enhance Analysis)

Adding these channels enables driver input analysis and basic vehicle dynamics

Throttle

Throttle position

Accepted Names:

Throttle Throttle Pos TPS

Unit: 0-100% or 0-1

Enables: Throttle application analysis

Brake

Brake pressure or position

Accepted Names:

Brake Brake Pressure Brake Pres Front

Unit: bar, psi, or %

Enables: Braking analysis, trail braking

RPM

Engine speed

Accepted Names:

RPM Engine RPM EngineSpeed

Unit: Revolutions per minute

Enables: Shift point analysis

Gear

Current gear

Accepted Names:

Gear Gear Position Gear Pos

Unit: Number (1-8)

Enables: Gear usage optimization

Steering

Steering wheel angle

Accepted Names:

Steering Steering Angle Steered Angle

Unit: Degrees

Enables: Steering smoothness analysis

Lap Time

Current lap time

Accepted Names:

Lap Time LapTime Running Lap Time

Unit: Seconds

Enables: Sector time analysis

Advanced Channels (Professional Analysis)

Professional telemetry channels for comprehensive vehicle dynamics analysis

G-Force Sensors

Lateral G

Side-to-side forces

Names: G Force Lat, Lateral G

Range: ±3.0 G typical

Longitudinal G

Acceleration/braking forces

Names: G Force Long, Longitudinal G

Range: ±2.0 G typical

Vertical G

Vertical forces

Names: G Force Vert, Vertical G

Range: 0.5-2.0 G typical

Enables: Traction circle analysis, grip utilization, weight transfer calculations

Suspension Sensors (Damper Potentiometers)

Front Left

Names: Susp Pos FL

Unit: mm

Front Right

Names: Susp Pos FR

Unit: mm

Rear Left

Names: Susp Pos RL

Unit: mm

Rear Right

Names: Susp Pos RR

Unit: mm

Enables: Ride height analysis, pitch/roll calculations, platform control, bottoming detection

Tire Temperature Sensors (3-Zone)

Inner Zone

FL: Tyre Temp FL Inner

FR: Tyre Temp FR Inner

Range: 60-120°C

Center Zone

FL: Tyre Temp FL Centre

FR: Tyre Temp FR Center

Optimal: 85-100°C

Outer Zone

FL: Tyre Temp FL Outer

FR: Tyre Temp FR Outer

Spread: <10°C ideal

Enables: Camber optimization, pressure adjustment, wear analysis

Setup Clues: Outer > Inner = too much negative camber | Center > Edges = over-inflated

Pro Tips

  • Channel names are flexible: The system will detect variations like "Throttle", "Throttle Pos", "TPS", etc.
  • Units are auto-detected: Use km/h or mph for speed, degrees or radians for angles
  • More channels = better analysis: Each additional channel unlocks new insights
  • Sample rate matters: 10-100 Hz is ideal for most analysis
  • Both laps should have same channels: Reference and current lap must match
